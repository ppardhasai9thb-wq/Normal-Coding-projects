# 1.Restaurant Menu Billing System

A simple console-based **Restaurant Menu Billing System** developed in **C++**. This project allows users to browse a restaurant menu, add food items to their order, and generate the final bill through an interactive command-line interface.

> **Note:** This project is part of my **Normal Coding Projects** repository, which contains multiple beginner-friendly projects demonstrating fundamental programming concepts.

---

## Features

- Interactive menu-driven interface
- Displays the price of selected food items
- Option to add items to the bill
- Supports ordering multiple items
- Calculates and displays the final bill
- Handles invalid menu selections

## How It Works

1. The restaurant menu is displayed.
2. The user selects an item from the menu.
3. The price of the selected item is shown.
4. The user chooses whether to add the item to the bill.
5. The user can continue ordering additional items.
6. Once finished, the program displays the total bill.
